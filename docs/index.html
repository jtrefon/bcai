Installation Instructions

Linux Installation

  1. Download the Linux package above
  2. Extract: tar -xzf bcai-linux-x64.tar.gz
  3. Run installer: cd bcai-linux-x64 && ./install.sh
  4. Verify: bcai --help

macOS Installation

  1. Download the appropriate macOS package (Intel or Apple Silicon)
  2. Extract: tar -xzf bcai-macos-*.tar.gz
  3. Run installer: cd bcai-macos-* && ./install.sh
  4. You may need to allow the app in System Preferences > Security
  5. Verify: bcai --help

Windows Installation

  1. Download the Windows package above
  2. Extract the ZIP file
  3. Run install.bat as administrator
  4. Open a new Command Prompt and verify: bcai --help
